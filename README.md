# RobotZoo.jl

Provides a handful of dynamics models of common robotic platforms. The following models
are currently implemented:
* Acrobot
* Dubins Car (unicycle model)
* Cartpole
* Double Integrator
* Pendulum
* Quadrotor
* Airplane
